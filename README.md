## Colour!
This is just a small C++ thingy to try and print stuff in colour to a terminal.  

Note that this does require a terminal (emulator) that supports 24-bits colour. (I'm using gnome-terminal)  

###### Building:
``g++ *.cpp -std=c++11 -o ./a.out``

###### Running: 
``./a.out``
